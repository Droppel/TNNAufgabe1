To start the program execute "python main.py" In the folder where "main.py", "neuralnet.py" and "PA-A-train.txt" are located.
The program then creates a "weights.txt" file with the final weights used by the perceptron.
It also prints its total square Error to stdout.

By Uncommenting lines 74-76 and commentine line 71 one can instead let the program read the final weights from "weights.txt".
It then prints the total square mean error to stdout.
